Plotfiles and Other Output

The following inputs must be preceded by “amr” and control the frequency and naming of plotfile generation, as well as whether a plotfile should be written out immediately after restarting a simulation.

Description

Type

Default

plot_int

Frequency of plotfile output; if -1 then no plotfiles will be written

Int

-1

plotfile_on_restart

Should we write a plotfile when we restart (only used if plot_int>0)

Bool

False

plot_file

Prefix to use for plotfile output

String

plt
