GPUÂ¶

In this chapter, we will present the GPU support in AMReX. Currently AMReX only supports Nvidia GPUs. Internally, it uses CUDA C++, but the users can use CUDA Fortran, OpenMP, and/or OpenACC. A recent version of CUDA (e.g., >= 9) is required, and the device must have compute capability >= 6.

For complete details of CUDA, CUDA Fortran, OpenMP and OpenACC languages, see their respective documentations.

A number of tutorials can be found at Tutorials/GPU/.
