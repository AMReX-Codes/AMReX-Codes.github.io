This routine is intended to be a generic fill function for cell centered data. It knows how to exrapolate, and reflect data and can be used to suppliment problem specific fill functions (ie. EXT_DIR).

Parameters
q<= array to fill
q_l1,q_l2,q_l3,q_h1,q_h2,q_h3=> index extent of q array
domlo,hi=> index extent of problem domain
dx=> cell spacing
xlo=> physical location of lower left hand corner of q array
bc=> array of boundary flags bc(SPACEDIM,lo:hi)

NOTE: corner data not used in computing soln but must have reasonable values for arithmetic to live
