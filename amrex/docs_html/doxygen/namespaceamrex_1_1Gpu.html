◆ freeAsync()

void amrex::Gpu::freeAsync ( Arena *  arena,
void *  mem 
)
inlinenoexcept

Deallocate memory belonging to an arena asynchronously. Memory deallocated in this way is held in a pool and will not be reused until the next amrex::Gpu::streamSynchronize(). GPU kernels that were already launched on the currently active stream can still continue to use the memory after this function is called. There is no need to use this function for CPU-only memory or withThe_Async_Arena.

Parameters
[in]arenathe arena the memory belongs to
[in]mempointer to the memory to be freed
