Detailed Description

template<int NStructReal, int NStructInt, int NArrayReal, int NArrayInt>
struct amrex::ParticleInitType< NStructReal, NStructInt, NArrayReal, NArrayInt >

This struct is used to pass initial data into the various Init methods of the particle container. That data should be initialized in the order real struct data, int struct data, real array data, int array data. If fewer components are specified than the template parameters specify for, a given component, then the extra values will be set to zero. If more components are specified, it is a compile-time error.

Example usage: 

ParticleInitType<0, 2, 4, 1> pdata = {{}, {7, 9}, {1.5, 2.5, 3.5, 4.5}, {11}};
