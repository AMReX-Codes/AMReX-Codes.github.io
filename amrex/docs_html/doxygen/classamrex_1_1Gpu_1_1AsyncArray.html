Detailed Description

template<typename T>
requires (std::is_trivially_copyable_v<T>)
class amrex::Gpu::AsyncArray< T >
Warning
AsyncArray assumes the GPU launch-region state stays fixed for its lifetime. Flipping the launch-region flag between construction and destruction prevents the matching free and will leak memory.
