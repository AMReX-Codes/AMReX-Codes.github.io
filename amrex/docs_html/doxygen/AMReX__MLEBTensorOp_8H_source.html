8 // Tensor solver for high Reynolds flows with small gradient in viscosity.
9 // The system it solves is
10 //
11 // alpha a v - beta div dot tau = rhs
12 //
13 // where tau = eta [grad v + (grad v)^T] + (kappa-(2/3)eta) (div v) I.
14 // Here eta and kappa are shear and bulk viscosity, and I is identity tensor.
15 //
16 // The user needs to provide `a` by `setACoeffs`, eta by `setShearViscosity`,
17 // and kappa by `setBulkViscosity`. If `setBulkViscosity` is not called,
18 // kappa is set to zero. The user must also call `setEBShearViscosity` to set
19 // viscosity on EB. Optionally, `setEBBulkViscosity` can be used to set
20 // bulk viscosity on EB.
21 //
22 // The scalars alpha and beta can be set with `setScalar(Real, Real)`. If
23 // they are not set, their default value is 1.
